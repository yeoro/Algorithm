### 알고리즘 분류
>- 브루트포스 알고리즘

### 풀이 시간
>- ⏱ 33분

### 해결
> ![good](../../../Img/good.png)
>1. 화단 가장자리를 제외한 부분 중에서 3곳을 골라 꽃을 심어야 하므로 완탐을 진행했다.
>2. 3곳을 선택했다면 각 지점에서 꽃을 피운다. 만약 이미 꽃잎이 존재하는 자리라면 꽃 피우는 작업을 중단한다. 꽃을 피우면서 동시에 화단의 가격도 계산한다. 
>3. 3개의 꽃을 모두 피웠다면, 현재 최솟값과 비교하여 최솟값을 갱신한다.
