### 해결

>1. 크루스칼 알고리즘 이용
>2. 배열에 각 별의 좌표를 입력한다.

>4. 우선순위 큐에서 간선 정보를 꺼내며 union-find 연산을 해주고, 거리를 구한 후 출력한다.
