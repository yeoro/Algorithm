> https://softeer.ai/practice/info.do?eventIdx=1&psProblemId=583

### 알고리즘 분류
>- BFS

### 풀이 시간
>- ⏱ 

### 해결
>1. 소나기, 태범 2개의 큐로 BFS 진행
>2. 태범이 큐가 빌 때까지, 각각의 큐 사이즈만큼 이동 진행. 이때 소나기 먼저 진행
>3. 소나기는 이동하면서 맵의 상태만 바꾸고, 태범이는 이동하면서 방문체크 상태만 바꾸게 했다.
