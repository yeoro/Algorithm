### 알고리즘 분류
>- 구현

### 풀이 시간
>- ⏱ 40분

### 채점 결과
>- 정확성: 100.0
>- 합계: 100.0 / 100.0

### 해결
>1. 키패드 배열을 만들어 각 번호의 좌표를 저장한다. *와 #는 각각 10, 11로 치환하여 좌표를 저장한다.
>2. 두 손가락의 위치를 저장할 배열을 선언한다. 왼손은 10번, 오른손은 11번의 좌표로 초기화해준다.
>3. numbers 배열에 있는 번호를 누르는 과정을 진행한다.
