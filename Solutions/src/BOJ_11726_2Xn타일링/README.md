### 문제 링크
> https://www.acmicpc.net/problem/11726

### 알고리즘 분류
>- 다이나믹 프로그래밍

### 풀이 시간
>- ⏱ 15분

### 해결
> ![bad](../../../Img/bad.png)  
>1. dp 배열의 값을 출력할 때 10,007로 나누는 것이 아니라 계산하여 저장할 때 10,0007로 나누어 저장해야 한다.
>  
> ![good](../../../Img/good.png)
>1. 오른쪽 끝에 1개의 타일을 추가하는 경우, 2개의 타일을 추가하는 경우 2가지 쌍의 경우로 타일을 놓을 수 있다. 
>2. 위를 점화식으로 나타내면 dp[n] = dp[n-1] + dp[n-2] 이 된다.
>3. 이 방법은 참고한 풀이에서 본 방식이고, 직접 타일을 그리면서 생각해보니 초기값이 다른 피보나치라는걸 알 수 있었다.
