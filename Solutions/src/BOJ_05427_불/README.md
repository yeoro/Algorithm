### 문제 링크
> https://www.acmicpc.net/problem/5427

### 알고리즘 분류
>- 그래프 이론
>- 그래프 탐색
>- 너비 우선 탐색

### 풀이 시간
>- ⏱ 2시간+

### 해결
> 1. **상근이**와 **불**을 관리할 **큐**를 **2**개 선언해야 한다.
> 2. **상근이 큐의 사이즈가 0일 때까지** 불 큐의 **사이즈 만큼 BFS 탐색**을 진행하고 난 후 상근이 큐의 **사이즈 만큼 BFS 탐색**을 진행하는 것이 핵심!
